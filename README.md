# 액션형 광고 카드 태그 디자인 제안

CPA/CPI/CPE 광고 카드의 태그를 **정보성 표기**와 **광고성(행동) 태그**로 명확히 구분하는 디자인 제안입니다.

🔗 **미리보기**: https://buzzviltheo.github.io/action-tag-design-test/

## 배경

포인트허브 PR [Buzzvil/pointhub-web#10108](https://github.com/Buzzvil/pointhub-web/pull/10108)은 광고명의 `[대괄호]` 텍스트를 추출해 글자 해시 기반 8색으로 칠합니다. 색은 일관되지만 **색과 의미가 연결되지 않아**, 유저는 "선착순"(정보성)과 "설치하면"(광고성)을 구분할 수 없습니다.

## 제안 — 두 종류를 자리부터 분리

| | 광고성 — 행동 후킹 태그 | 정보성 표기 |
|---|---|---|
| 예 | 설치하면 · 사면 적립 · 간편가입 · 플레이만 | 선착순 · 할수록 적립 |
| **위치** | 텍스트(subtitle) 라인 — 참여 방법 | 썸네일 코너 — 광고 속성 |
| **색** | 브랜드 레드 채움 | 뉴트럴 칩 |
| **형태** | 솔리드 | 반투명 칩 + 아이콘 |
| **부여** | 라인아이템별 데이터 매핑 | category 런타임 식별(매핑 X) |

**위치 + 색 + 형태 + 아이콘** 4중 구분이라, 같은 자리에서 색만 다르던 기존 방식과 달리 자리만 봐도 종류가 읽힙니다. 한 광고에 둘 다 있어도(선착순 + 사면 적립) 자리가 달라 부딪히지 않습니다.

## 노출 원칙

- **자리로 역할 분리** — 정보성은 썸네일, 행동은 텍스트
- **가속 소진(ASAP) 광고에만** 행동 태그 노출 (변별력 유지)
- **광고명 `[대괄호]` 제거** 후 우리 태그 체계로 표기

## 문구 원칙 (토스애즈 UX라이팅 5원칙 검토)

태그는 5자 라벨이라 "행동 가볍게 · 쉬운 말"만 태그 몫이고, "확실한 보상 · 구체 숫자"는 CTA 버튼·부제목이 담당합니다. 무거운 행동(가입·상담·신청)은 "간편-" 톤으로 완화하되, 액션 명확성을 해치는 과한 경량화는 지양합니다.
