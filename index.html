액션형 광고 카드 태그

광고가 요구하는 행동 태그와 광고의 성격을 알리는 정보성 표기를 위치까지 분리해 한눈에 구분합니다. 정보성은 썸네일 코너(광고 속성), 행동 태그는 텍스트 라인(참여 방법). 카드당 절제해서 노출해 노이즈를 줄입니다.

01 — 태그는 두 종류, 자리도 다르다
광고성 · Action

행동 후킹 태그

유저가 무엇을 하면 보상받는지. 참여 방법 안내 → 텍스트(subtitle) 라인.

설치하면 사면 적립 간편가입 플레이만 간편조회
  • 위치 텍스트 라인 (참여 방법 자리)
  • 색 브랜드 레드 — 행동 유도 강조
  • 형태 솔리드 채움
  • 부여 라인아이템별 데이터 매핑
정보성 · Info

정보성 표기

광고 성격을 알리는 속성. 행동 유도가 아닌 안내 → 썸네일 코너.

선착순 할수록 적립
  • 위치 썸네일 코너 (광고 속성 자리)
  • 색 뉴트럴 — 차분한 보조
  • 형태 반투명 칩 + 아이콘
  • 부여 매핑 안 함 · category 런타임 식별
위치 + 색 + 형태 + 아이콘으로 구분합니다. 같은 자리에서 색만 다르던 기존 방식과 달리, 정보성은 썸네일·행동은 텍스트로 역할이 자리에서 드러나 한 광고에 둘 다 있어도(선착순 + 사면 적립) 부딪히지 않습니다.
02 — 현재 방식의 문제
현재 — 해시 8색, 같은 자리
선착순 설치하면 멀티미션!

대괄호 텍스트를 글자 해시로 색만 칠해 같은 자리에 표시합니다. 색과 의미가 연결되지 않고 종류 구분도 위치 구분도 없어, 유저는 무엇이 행동이고 무엇이 정보인지 알 수 없습니다.

개선 — 의미 기반, 자리 분리
선착순 (썸네일) 설치하면 (텍스트)

정보성은 썸네일 코너, 행동은 텍스트 라인. 자리가 역할을 말합니다. 종류가 둘뿐이라 카드가 정돈됩니다.

04 — 노이즈를 줄이는 노출 원칙
01

자리로 역할 분리

정보성은 썸네일 코너, 행동은 텍스트 라인. 위치가 종류를 말해줘 색·형태에만 기대지 않습니다.

02

가속 소진만 노출

행동 태그는 가속 소진(ASAP) 광고에만. 모든 카드에 붙으면 변별력이 사라지므로 절제.

03

대괄호 제거

광고명의 [대괄호]는 모두 제거하고 깔끔한 광고명만. 태그는 우리 체계로 따로 표기.

05 — 태그 문구 원칙

태그는 5자 라벨이라 카피 전체 원칙을 다 담지 않습니다. 행동을 가볍게·쉬운 말로가 태그의 몫이고, 확실한 보상·구체 숫자는 CTA 버튼·부제목이 맡습니다.

·

행동을 가볍게

"~하면·~만·보면"처럼 부담 낮은 동사. 무거운 행동(가입·상담·신청)은 "간편-" 톤으로.

·

쉬운 말로

자주 쓰는 자연스러운 한국어. 억지 축약·외래어 지양. ("구경하면" 등)

·

보상은 버튼에

금액·할인율·확실성은 태그가 아니라 CTA 버튼·부제목이 담당. 태그에 욱여넣지 않음.
